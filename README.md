## Getting Started âœ¨
#### Prerequisites
* Only Node.js v14.X or newer can be used for this project.
* You also need your bot's token. Check this <a href="https://anidiots.guide/getting-started/getting-started-long-version">Guide</a> for more informations.

#### Configuration
Open the `app.ts` in the `src` directory. There you can set your token, change the prefix for the commands and adjust the streams.
`âš ï¸ The data in the app.ts should not be published publicly otherwise third parties can gain access to your discord bot.`

#### Installation:
1. Download the github repository.
```
git clone https://github.com/VocalZero/discordjs-musicbot
````
2. Navigate in the project folder and install all dependencies.
```
cd discordjs-musicbot
npm install
````
3. After the installation you can start the bot.
```
npm run start
````
